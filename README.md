# SCP-JP API

[SCP Crawler](https://github.com/scp-data/scp_crawler)を基に[SCP財団Wiki 日本語版(SCP-JP)](http://scp-jp.wikidot.com/)に対応したものです。

このリポジトリはSCPデータを構築し、格納しています。

[SCP-JP Crawler](https://github.com/SCP-JP-Crawler/scp-jp-crawler)プロジェクトとGitHub Actionsを使用して、SCPデータセットをGitHub Pagesに公開します。このリポジトリをクローンすることで、データを直接利用することも可能です。

データは[docs](./docs)サブフォルダに格納されており、データ構造を説明するREADMEファイルが含まれています。
